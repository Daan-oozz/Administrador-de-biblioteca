# Historias de Usuario

1. Como administrador, quiero registrar un libro para que el sistema tenga nuevos ejemplares disponibles.
2. Como usuario, quiero buscar libros por título o autor para encontrar rápidamente lo que necesito.
3. Como usuario, quiero prestar un libro para llevármelo temporalmente.
4. Como usuario, quiero devolver un libro para actualizar el inventario.
5. Como administrador, quiero eliminar un libro para mantener el catálogo limpio y actualizado.
